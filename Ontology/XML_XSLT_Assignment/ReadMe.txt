1. Unzip all the files in same directory.
2. Open command window in the same directory.
3. Type the following command to run the .xsl file:

   msxsl.exe Assignment1.xml Assignment1.xsl -o ouput.html news="sports" category="tennis"
   
4. The parameter news="current_affairs" and place="New Delhi" can also be used.  
5. The parameter news="sports" and category="football" can also be used.
6. The parameter news="technology" and category="Mobile & Tabs" can also be used.
7. You can skip the place and category part to list all the news related to either sports or 
   current_affairs or tecnology.
